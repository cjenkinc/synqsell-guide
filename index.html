Getting Started with Synqy

Welcome to Synqy, your Discord bot for seamless product purchases, queue management, and community fun! Follow these steps to get started:

  • Configure Settings: Server admins can set shipping rates and timers using `/set-shipping-rates` and `/set-givvy-timer`.
  • Start Using Commands: Check out the commands below to buy products, manage tabs, and view queues.

Common User Commands

These commands are available to all users in servers where Synqy is invited:

/buy - Add items to your tab by selecting a product via autocomplete.

Example: Type `/buy` and select `mtg_2x2_jpcb` to buy it, then confirm the quantity in the interactive menu.

/tabs - View your current tab (purchase total) in the server.

Output: Shows your total owed, e.g., “Your tab is $60.00.”

/list-products - List all available products and their prices.

Output: Shows products like “- `mtg_2x2_jpcb`: $60.00”.

/search-product - Search for products by name or part of the name.

Example: `/search-product mtg` lists Magic: The Gathering products.

/queue - View the current buy queue (admin-only, but useful for users to know).

Output: Displays queued purchases, e.g., “- @User purchased 1x `mtg_2x2_jpcb` (ID: ).”

Admin-Only Commands

Server administrators can use these commands to manage Synqy:

/load-data - Load product data from a CSV file with columns 'Product' and 'Price' (requires Administrator).

Example: Upload a CSV with the following format to add products:

Product,Price mtg_2x2_jpcb,60.00 mtg_blb_jp,22.00 poke_prismatic_etb,100.00

Notes: Save your CSV file (e.g., `products.csv`) and upload it using the Discord attachment feature with this command.

/remove-product - Remove a product by name (requires Administrator).

Example: `/remove-product mtg_2x2_jpcb` removes that product.

/remove-from-queue - Remove an item from the queue by ID (requires Administrator).

Example: `/remove-from-queue 1` removes item ID 1, adjusting the user’s tab.

/move-in-queue - Move an item to a new position in the queue (requires Administrator).

Example: `/move-in-queue 1 2` moves item 1 to position 2.

/process-queue - Process the next item in the buy queue (requires Administrator).

Output: Notifies the user and moves to the next in queue, e.g., “Processed queue item for User! 1x `mtg_2x2_jpcb`.”

/set-shipping-rates - Set shipping limits and costs for the server (requires Administrator).

Example: `/set-shipping-rates 50 5` sets a $50 limit and $5 cost.

/set-givvy-timer - Set the givvy timer for voice channel giveaways (requires Administrator).

Example: `/set-givvy-timer 30` sets a 30-minute timer.
